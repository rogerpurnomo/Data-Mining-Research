# Data-Mining-Research

📊 Optimizing Social Commerce using Association Rule Mining
A data mining project that applies Association Rule Mining (ARM) and Market Basket Analysis (MBA) to analyze purchasing behavior in TikTok Shop.

📌 Overview
This project explores how data mining techniques can improve:
🛍️ Customer shopping experience
📈 Vendor sales strategies
🤖 Product recommendation systems
The research focuses on TikTok Shop, a discovery-driven e-commerce platform, where user behavior is influenced by content and trends.

🎯 Problem Statement
- Users often make impulsive purchases without relevant recommendations
- Vendors struggle to target the right audience
- Lack of insights into product relationships and buying patterns

👉 Example rule:
If a user buys Product A, they are likely to buy Product B

💡 Key Insights
- Identified frequent product combinations
- Improved cross-selling and bundling strategies
- Enhanced personalized recommendation systems
- Reduced decision fatigue for users
